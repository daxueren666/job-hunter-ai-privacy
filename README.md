# Job Hunter AI 隐私政策
最后更新日期：2026年03月11日 

本扩展尊重并严格保护所有用户的个人隐私，本政策将详细说明我们对用户数据的处理方式，请您仔细阅读。

1. 数据收集与存储
本扩展不会通过任何方式收集、上传用户的个人数据至第三方服务器。
所有用户数据（包括但不限于AI API Key、简历内容、生成的求职内容、用户配置信息）均仅存储在用户本地设备的Chrome浏览器 chrome.storage 中，完全由用户自行掌控。

2. 数据使用
用户提供的所有数据，仅用于实现扩展的核心功能（包括打招呼语生成、面试题预测、简历优化），不会用于任何其他用途。
扩展的AI功能仅通过用户自行配置的API Key，直接向对应AI服务商发起HTTPS加密请求，无任何第三方中转。

3. 数据共享
我们不会出售、共享、泄露用户的任何个人数据给任何第三方机构、组织或个人。
仅在法律法规强制要求的情况下，我们才会配合提供相关信息，除此之外无任何数据共享行为。

4. 用户权利
用户可随时通过扩展的设置页，一键清空所有本地存储的数据；也可通过卸载扩展，彻底删除所有存储在本地的内容，完全掌控自己的数据。

5. 权限说明
本扩展申请的所有浏览器权限，均仅用于实现扩展的核心功能，不会用于其他用途。

6. 政策更新
若本隐私政策有更新，我们会通过扩展更新公告的方式通知用户。

7. 联系方式
如有任何隐私相关的疑问，可通过评论区/私信联系我们。
